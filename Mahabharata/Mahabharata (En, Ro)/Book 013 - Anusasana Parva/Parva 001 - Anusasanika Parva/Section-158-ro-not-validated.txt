SECTIUNEA 158

"Yudhishthira a spus, "Tu întotdeauna te închini, O rege, Brahmanilor cu jurăminte lăudabile. Totuși, care este acel fruct pe care îl vezi și pentru care te închini lor, O rege? O, tu cu jurăminte înalte, văzând ce prosperitate se atașează închinării Brahmanilor, de ce te închini lor? Spune-mi totul, O, tu cu brațe puternice!"

"Bhishma a spus: 'Iată-l pe Kesava, înzestrat cu o mare inteligență. El îți va spune totul. Cu jurăminte înalte și înzestrat cu prosperitate, chiar el vă va spune ce este prosperitatea care se atașează de adorarea Brahmanilor. Puterea mea, urechile, vorbirea, mintea, ochii și acea înțelegere clară a mea (sunt toate înnorate astăzi). Cred că timpul nu este îndepărtat când va trebui să-mi arunc corpul. Soarele mi se pare că merge foarte încet.[611] Acele îndatoriri înalte, o rege, care sunt menționate în Purane ca fiind observate de Brahmani și Kshatriyas și Vaisyas și Sudras, au fost toate recitate de mine. Tu, o fiu al Pritha, învață de la Krishna ceea ce mai rămâne de învățat în acest sens. Îl cunosc pe Krishna adevărat. Știu cine este el și care este puterea lui antică. O șef al Kauravas, Kesava este de suflet nemăsurabil. Ori de câte ori apar îndoieli, este el cel care susține Dreptatea atunci.[612] Este Krishna care a creat pământul, cerul și cerurile. Într-adevăr, pământul a izvorât din corpul lui Krishna. De teribilă putere și existând de la începutul timpului, este Krishna care a devenit mistrețul puternic și a ridicat Pământul scufundat. Este El care a creat toate punctele cardinale, împreună cu toate munții. Sub El sunt cerul, cerul, cele patru puncte cardinale și cele patru puncte subsidiare. Este de la el că întreaga creație a curs. Este El care a creat acest univers antic. În buricul Lui a apărut un Lotus. În acel Lotus a izvorât însuși Brahma de energie nemăsurabilă. A fost Brahma, o fiu al Pritha, care a rupt acea întuneric care exista depășind chiar oceanul (în adâncime și întindere). În epoca Treta, o Partha, Krishna a existat (pe pământ), sub forma Dreptății. În epoca Treta, el a existat sub forma Cunoașterii. În epoca Dwapara, el a existat sub forma puterii. În epoca Kali a venit pe pământ sub forma nedreptății. Este El care în zilele de demult a ucis Daityas. Este El care este Zeul antic. Este El care a condus Asuras sub forma împăratului lor (Valin). Este El care este Creatorul tuturor ființelor. Este El care este și viitorul tuturor ființelor create. Este El care este și protectorul acestui univers însărcinat cu sămânța distrugerii. Când cauza Dreptății slăbește, acest Krishna se naște în rasa fie a zeilor, fie printre oameni. Stând pe Dreptate, acest Krishna de suflet curățat (cu astfel de ocazie) protejează atât lumea superioară, cât și cea inferioară. Păstrându-i pe cei care merită să fie cruțați, Krishna se pregătește pentru masacrul Asura, o Partha! Este el care este toate actele corecte și necorespunzătoare și este el care este cauza. Este Krishna care este actul făcut, actul care trebuie făcut și actul care se face. Știți că acel ilustru este Rahu și Soma și Sakra. Este el care este Viswakarma. Este el care este de formă universală. El este distrugătorul și el este Creatorul universului. El este cel care mânuiește Sula (lancea); El este de formă umană; și El este de formă teribilă. Toate creaturile îi cântă laudele, pentru că este cunoscut prin faptele sale. Sute de Gandharvas și Apsaras și zeități îl însoțesc întotdeauna. Chiar și Rakshasas îi cântă laudele. El este Îmbunătățitorul Bogăției; El este singura Ființă victorioasă din univers. În Sacrificii, oamenii elocvenți îi cântă laudele. Cântăreții de Samans îl laudă recitând Rathantaras. Brahmanii îl laudă cu Mantras Vedice. Este către El că preoții sacrificatori își varsă libațiile. Zeitățile cu Indra la capul lor i-au cântat laudele când a ridicat munții Gobardhana pentru a proteja păstorii de vaci din Brindavana împotriva ploilor neîncetate pe care Indra le-a turnat în furie. El este, o Bharata, o singură Binecuvântare pentru toate creaturile. El, o Bharata, intrând în vechea peșteră Brahma, a văzut de acolo acoperișul original al lumii la începutul Timpului.[613] Agitând toți Danavas și Asuras, acest Krishna de fapte de frunte a salvat pământul. Este către El că oamenii dedică diverse tipuri de mâncare. Este către El că războinicii dedică toate tipurile de vehiculele lor în timpul războiului. El este etern, și este sub acel ilustru că cerul, pământul, cerul, toate lucrurile există și rămân. Este el care a cauzat sămânța vitală a zeilor Mitra și Varuna să cadă într-un borcan, de unde a izvorât Rishi cunoscut sub numele de Vasishtha. Este Krishna care este zeul vântului; este El care este puternicul Aswins; este El care este acel primul dintre zei, adică soarele posedat de o mie de raze. Este El prin care Asuras au fost subjugate. Este El care a acoperit cele trei lumi cu trei pași ai Săi. El este sufletul zeităților și al ființelor umane, și Pitris. Este El care este Sacrificiul efectuat de acele persoane care sunt familiarizate cu ritualurile sacrificiilor. Este El care se ridică în fiecare zi în firmament (sub forma soarelui) și împarte Timpul în zi și noapte, și cursuri timp de jumătate de an spre nord și timp de jumătate de an spre sud. Numeroase raze de lumină emană de la El în sus și în jos și transversal și luminează pământul. Brahmanii familiarizați cu Vedele îl adoră. Luând o porțiune din razele Lui soarele strălucește în firmament. Lună după lună, sacrificatorul îl ordonă ca un sacrificiu. Persoanele regenerate familiarizate cu Vedele îi cântă laudele în sacrificii de toate felurile. Este el care constituie roata anului, având trei nave și șapte cai pentru a o trage. Este în acest fel că El susține tripla mansardă (a anotimpurilor). Înzestrat cu o mare energie, pătrunzând toate lucrurile, cel mai de seamă dintre toate creaturile, este Krishna care singur susține toate lumele. El este soarele, cel care alungă toate întunericurile. El este Creatorul tuturor. Tu, o erou, te apropie de acel Krishna! Odată, înaltul și puternicul Krishna a locuit, pentru o vreme, sub forma lui Agni în pădurea Khandava printre niște paie sau iarbă uscată. În curând a fost satisfăcut (pentru că a consumat toate plantele medicinale din acea pădure). Capabil de a merge oriunde la voință, a fost Krishna care, după ce a subjugat Rakshasas și Uragas, le-a turnat ca libații pe focul aprins. Este Krishna care i-a dat lui Arjuna un număr de cai albi. Este El care este creatorul tuturor cailor. Această lume (sau, viața umană) reprezintă carul său. Este el care înhamează acel car pentru a-l pune în mișcare. Acel car are trei roți (adică cele trei atribute ale Sattwa, Rajas și Tamas). Are trei tipuri de mișcare (pentru că merge în sus sau în jos sau transversal, implicând nașterea superioară, inferioară și intermediară, așa cum este adusă de acte). Are patru cai înhamate la el (adică Timp, Predestinare, voința zeităților și propria voință). Are trei nave (albe, negre și mixte, implicând fapte bune, fapte rele și fapte care sunt de un caracter mixt). Este acest Krishna care este refugiul celor cinci elemente originale cu cerul printre ele. Este El care a creat pământul și cerul și spațiul dintre. Într-adevăr, este acest Krishna de energie nemăsurabilă și strălucitoare care a creat pădurile și munții. Este acest Krishna care, dorind să-l pedepsească pe Sakra care era pe cale să-și arunce tunetul asupra lui, a trecut râurile și odată l-a paralizat. El este marea Indra care este adorată de Brahmani în mari sacrificii cu ajutorul a o mie de Riks vechi. A fost acest Krishna, o rege, care singur a fost capabil să păstreze Rishi Durvasa de mare energie ca oaspete pentru ceva timp în casa lui. Se spune că este Rishi antic. El este Creatorul universului. Într-adevăr, El creează totul din propria sa natură. Superior tuturor zeităților este El care învață toate zeitățile. El respectă cu strictețe toate ordonanțele antice. Știți, o rege, că acest Krishna, care este numit Vishwaksena, este fructul tuturor actelor care se referă la plăcere, a tuturor actelor care sunt întemeiate pe Vedele și a tuturor actelor care aparțin lumii. El este razele albe de lumină care se văd în toate lumele. El este cele trei lumi. El este cei trei Regenți ai tuturor lumilor. El este cele trei focuri de sacrificiu. El este cele trei Vyahritis; într-adevăr, acest fiu al Devaki este toți zeii împreună. El este anul; El este Anotimpurile; El este Fortnights; El este Ziua și Noaptea; El este acele diviziuni de timp care sunt numite Kalas, și Kashthas, și Matras, și Muhurtas, și Lavas, și Kshanas. Știți că acest Vishwaksena este toate acestea. Luna și Soarele, Planetele, Constelațiile și Stelele, toate zilele Parva, inclusiv ziua de lună plină, conjuncțiile constelațiilor și anotimpurile, au, o fiu al Pritha, izvorât din acest Krishna care este Vishwaksena. Rudras, Adityas, Vasus, Aswins, Sadhyas, Viswedevas, diversele Maruts, Prajapati însuși, mama zeităților, adică Aditi, și cei șapte Rishis, au izvorât toți din Krishna. Transformându-Se în Vânt, El împrăștie universul. De formă Universală, El devine Foc care arde toate lucrurile. Schimbându-Se în Apă, El înmoaie și submerge totul, și asumându-și forma lui Brahman, El creează toate triburile diverse de creaturi animate și inanimate. El este Însuși Veda, totuși el învață toate Vedele. El este Însuși toate ordonanțele, totuși El respectă toate ordonanțele care au fost stabilite în probleme legate de Dreptate și Vedele și acea forță sau putere care guvernează lumea. Într-adevăr, știți, o Yudhishthira, că acest Kesava este întregul univers mobil și imobil. El este de forma celei mai strălucitoare lumini. De formă universală, acest Krishna este afișat în acea strălucire strălucitoare. Cauza originală a sufletului tuturor creaturilor existente. El a creat mai întâi apele. Apoi a creat acest univers. Știți că acest Krishna este Vishnu. Știți că El este sufletul universului. Știți că El este toate anotimpurile; El este aceste vegetații diverse minunate ale Naturii pe care le vedem; El este norii care varsă ploaie și fulgerul care strălucește pe cer. El este elefantul Airavata. De fapt, El este întregul univers mobil și imobil. Locuința universului și transcendent toate atributele, acest Krishna este Vasudeva. Când devine Jiva, este numit Sankarshana. Apoi, El se transformă în Pradyumna și apoi în Aniruddha. În acest fel, înaltul Krishna, care are propria sa origine se împarte (sau se afișează) în formă patrulată. Dorind să creeze acest univers care constă din cele cinci elemente primordiale, El se pune la treabă și îl face să continue în forma cincinală a existenței animate constând din zeități și Asuras și ființe umane și fiare și păsări. Este el care atunci creează Pământul și Vântul, Cerul, Lumina și, de asemenea, Apa, o fiu al Pritha! După ce a creat acest univers de obiecte imobile și mobile distribuite în patru ordine de ființă (adică vivipare, ovipare, vegetale și născute din murdărie), el a creat apoi pământul cu sămânța sa cincinală. El a creat apoi firmamentul pentru a turna ploi abundente de apă pe pământ.[614] Fără îndoială, o rege, este acest Krishna care a creat acest univers. Originea lui este în propria sa persoană; este El care face ca toate lucrurile să existe prin propria sa putere. Este el care a creat zeitățile, Asuras, ființele umane, lumea, Rishis, Pitris și toate creaturile. Dorind să creeze, acel Domn al tuturor creaturilor a creat în mod corespunzător întregul univers al vieții. Știți că binele și răul, mobilul și imobilul, au curs toate din acest Unul care este Vishwaksena. Orice există și orice va apărea în existență, totul este Kesava. Acest Krishna este și moartea care îi surprinde pe toate creaturile când vine sfârșitul lor. El este etern și este El care susține cauza Dreptății. Orice a existat în trecut și orice nu știm, într-adevăr, totul este și acest Vishwaksena. Orice este nobil și meritoriu în univers, într-adevăr, orice de bine și de rău există, totul este Kesava care este de neconceput. Prin urmare, este absurd să te gândești la ceva care este superior lui Kesava. Kesava este chiar așa. Mai mult decât atât, El este Narayana, cel mai înalt dintre cei înalți, imuabil și nefadat. El este cauza eternă și imuabilă a întregului univers mobil și imobil cu începutul, mijlocul și sfârșitul său, precum și a tuturor creaturilor a căror naștere urmează dorința lor.'"



Note:

611. Pentru o persoană chinuită, ziua pare lungă.

612. Sensul este că acest Kesava este cel care susține cauza Dreptății atunci când pericolele o amenință, cf. 'Yada yada hi dharmasya, etc.' în Gita. Nu înseamnă că atunci când persoanele au îndoieli cu privire la chestiuni de moralitate, este Kesava cel care le înlătură.

613. Se referă la existența lui Brahma când totul altceva nu există.

614. Semința cvintuplă constă în cele patru ordine de creaturi și acțiuni care determină condițiile tuturor ființelor.
