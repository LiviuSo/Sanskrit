SECTIUNEA 254

"Vyasa a spus: 'Există un copac minunat, numit Dorință, în inima unui om. Acesta se naște din sămânța numită Eroare. Mânia și mândria constituie trunchiul său mare. Dorința de acțiune este bazinul din jurul piciorului său (pentru a ține apa care urmează să îl hrănească). Ignoranța este rădăcina acelui copac, iar neglijența este apa care îi dă subzistență. Invidia constituie frunzele sale. Faptele rele din viețile anterioare îi oferă vigoare. Pierderea judecății și anxietatea sunt ramurile sale; durerea formează ramurile sale mari; iar frica este lăstarul său. Setea (după diverse obiecte) care este (aparent) plăcută formează vițele care se învârt în jurul său pe toate părțile. Bărbații extrem de lacomi, legați în lanțuri de fier, stând în jurul acelui copac care dă fructe, îi aduc venerație, în așteptarea obținerii fructului său. Cel care, înfruntând acele lanțuri, taie acel copac și caută să scape atât de tristețe cât și de bucurie, reușește să ajungă la sfârșitul ambelor. Acel om prost care hrănește acest copac prin indulgență în obiectele simțurilor este distrus de acele obiecte în care se complac, după modul în care o pilulă otrăvitoare distruge pacientul căruia i se administrează. O persoană pricepută, cu toate acestea, cu ajutorul Yoga, smulge cu forța și taie cu sabia samadhi, rădăcina îndepărtată a acestui copac. Cel care știe că sfârșitul tuturor actelor întreprinse doar din dorința de fruct este renașterea sau lanțurile care leagă, reușește să depășească toată durerea. Corpul este considerat a fi un oraș. Înțelegerea este considerată a fi stăpâna sa. Mintea care locuiește în corp este ministrul acelei stăpâne a cărei funcție principală este să decidă. Simțurile sunt cetățenii care sunt angajați de minte (în serviciul stăpânei). Pentru a îngriji acei cetățeni, mintea afișează o înclinație puternică pentru acte de diverse feluri. În ceea ce privește acele acte, se observă două mari defecte, și anume Tamas și Rajas. Asupra fructelor acelor acte se odihnesc acei cetățeni împreună cu șefii orașului (adică Mintea, Înțelegerea și Conștiința). Cele două defecte (deja menționate) trăiesc pe fructele acelor acte care sunt realizate prin mijloace interzise. Aceasta fiind situația, înțelegerea, care în sine este de neînvins (de Rajas sau Tamas), coboară la o stare de egalitate cu mintea (devenind la fel de pătată ca mintea care o servește). Apoi, simțurile, agitate de mintea pătată, își pierd propria stabilitate. Acele obiecte pentru a căror achiziție înțelegerea se străduiește (considerându-le benefice) devin productive de durere și în cele din urmă se întâlnesc cu distrugerea. Acele obiecte, după distrugere, sunt reamintite de minte, și în consecință ele afectează mintea chiar și după ce sunt pierdute. Înțelegerea este afectată în același timp, pentru că mintea este considerată diferită de înțelegere doar atunci când mintea este considerată în ceea ce privește funcția sa principală de a primi impresii despre a căror certitudine nu este judecător. În realitate, totuși, mintea este identică cu înțelegerea. Rajas (productiv doar de durere și rău de orice fel) care este în înțelegere atunci copleșește sufletul însuși care se află peste înțelegerea pătată de Rajas ca o imagine pe o oglindă. Este mintea care se unește prima dată în prietenie cu Rajas. După ce s-a unit, ea capturează sufletul, înțelegerea și simțurile (ca un ministru fals care capturează regele și cetățenii după ce a conspirat cu un dușman) și le face pe toate peste Rajas (cu care s-a unit).'"


Note:

1096. Citirea corectă este ayasaih, care înseamnă 'făcut din fier', și nu 'ayasaih'. K.P. Singha aderă la citirea incorectă. Lanțurile de fier de aici sunt fie dorințele diverse pe care le nutresc oamenii lumii, fie, poate, corpurile cu care oamenii sunt înzestrați.

1097. Dualul genitiv duhkhayoh este folosit deoarece sukha lumesc este de asemenea considerat ca duhkha. 'Tyajamannah' este echivalent cu 'tyaktum ichccha.' Este un exemplu de hetau sanach.

1098. Yena este explicat ca Stryadina hetuna. 'Sah' este Stryadih. Samrohati este Vardhayati. 'Tam' este Vardhakam.

1099. 'Uddhriyate' se traduce literal prin 'rupere'. Utilizarea cuvântului 'asina' sugerează, de asemenea, 'tăiere'. Rădăcina copacului, desigur, este Avidya sau Ignoranță.

1100. K.P. Singha traduce greșit prima linie. Traducătorul din Burdwan citează glosa fără a o înțelege. Prima jumătate a primei linii, tradusă literal, este 'simțurile sunt cetățenii minții', adică, după cum explică corect comentatorul, sunt cetățeni sub conducerea minții. 'Tadartham' înseamnă 'pentru binele simțurilor', adică 'pentru a le îngriji'. Prakritih este mahati kriya pravrittih, Tadartham este kriyaphalam, adică fericire sau suferință. Sensul, în rezumat, este acesta: corpul este un oraș. Înțelegerea este stăpâna sa. Mintea este principalul său servitor. Simțurile sunt cetățenii sub conducerea minții. Pentru a îngriji simțurile, mintea se angajează în acte care produc fructe vizibile și invizibile, adică sacrificii și daruri, și achiziționarea de case și grădini, etc. Aceste acte sunt supuse la două defecte, și anume, Rajas și Tamas. Simțurile (atât în această viață cât și în cele viitoare) depind de fructele (fericire sau suferință) ale acestor acte.

1101. Sensul este următorul: simțurile, mintea, înțelegerea, etc., sunt toate datorate acțiunilor. Prin urmare, se spune că acestea se bazează pe acțiuni și își trag hrana de acolo.

1102. Extind prima linie de 14 pentru a explica clar semnificația.

1103. Sensul este că înțelegerea, fiind pătată sau afectată, sufletul devine de asemenea pătat sau afectat. Enam este atmanam. Vidhritam este 'plasat ca o imagine pe o oglindă'.
