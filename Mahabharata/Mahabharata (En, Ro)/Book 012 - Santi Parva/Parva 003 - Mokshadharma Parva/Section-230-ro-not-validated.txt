SECTIUNEA 230

"Yudhishthira a spus, "Ce om există care este drag tuturor, care îi înveselește pe toți și care este înzestrat cu fiecare merit și fiecare realizare?"

"Bhishma a spus: "În această legătură, îți voi recita cuvintele pe care Kesava, întrebat de Ugrasena, i le-a spus cu ocazia unei întâlniri anterioare.

"Ugrasena a spus: 'Toate persoanele par a fi foarte dornice să vorbească despre meritele lui Narada. Cred că acel Rishi celestial trebuie să fie cu adevărat înzestrat cu orice fel de merit. Îți cer, spune-mi asta, O Kesava!'"

"'"Vasudeva a spus: 'O, șefule al Kukkuras, ascultă-mă pe măsură ce menționez în mod succint acele calități bune ale lui Narada cu care sunt familiarizat, o, rege! Narada este la fel de învățat în scripturi pe cât este de bun și pios în comportamentul său. Și totuși, din cauza comportamentului său, el nu își cultivă niciodată mândria care îți face sângele atât de fierbinte. Din acest motiv, el este venerat peste tot. Nemulțumirea, mânia, ușurința și frica, acestea nu există în Narada. El este liber de amânare și posedă curaj. Pentru asta este venerat peste tot. Narada merită respectul tuturor. El nu se abate niciodată de la cuvintele sale din dorință sau lăcomie. Pentru asta este venerat peste tot. El este pe deplin familiarizat cu principiile care conduc la cunoașterea sufletului, predispus la pace, posedă o energie mare și este stăpân pe simțurile sale. El este liber de viclenie și spune adevărul. Pentru asta este venerat cu respect peste tot. El se distinge prin energie, faimă, inteligență, cunoaștere, umilință, naștere, penitențe și ani. Pentru acestea este venerat cu respect peste tot. El are un comportament bun. Se îmbracă și se găzduiește bine. Mănâncă mâncare curată. Iubește pe toți. Este curat în trup și minte. Vorbește dulce. Este liber de invidie și răutate. Pentru asta este venerat cu respect peste tot. Cu siguranță este întotdeauna angajat în a face bine tuturor oamenilor. Niciun păcat nu locuiește în el. El nu se bucură niciodată de nenorocirile altor oameni. Pentru asta este venerat cu respect peste tot. El caută întotdeauna să cucerească toate dorințele pământești ascultând recitările vedice și participând la Purane. El este un mare renunțător și nu disprețuiește pe nimeni.[869] Pentru asta este venerat cu respect peste tot. El privește cu un ochi egal pe toți; și, prin urmare, nu are pe nimeni pe care îl iubește și nici pe cine urăște. El vorbește întotdeauna ceea ce este plăcut ascultătorului. Pentru asta este venerat cu respect peste tot. El posedă o mare învățătură în scripturi. Conversația lui este variată și încântătoare. Cunoștințele și înțelepciunea lui sunt mari. Este liber de lăcomie. Este liber și de înșelăciune. Este generos. A învins mânia și lăcomia. Pentru asta este venerat cu respect peste tot. El nu s-a certat niciodată cu nimeni pentru niciun subiect legat de profit sau plăcere. Toate defectele au fost îndepărtate de el. Pentru asta este venerat cu respect peste tot. Devotamentul lui (pentru Brahma) este ferm. Sufletul lui este fără vină. El este bine versat în Srutis. Este liber de cruzime. Este dincolo de influența iluziei sau a defectelor. Pentru asta este venerat peste tot cu respect. El nu este atașat de toate acele lucruri care sunt obiecte de atașament (pentru alții). Pentru toate acestea, el pare a fi atașat de toate lucrurile.[870] El nu este niciodată supus influenței îndoielii pentru mult timp. Pentru asta este venerat peste tot cu respect. El nu are nicio dorință pentru obiecte legate de profit și plăcere. El nu își glorifică niciodată propria persoană. Este liber de răutate. Este blând în vorbire. Pentru asta este venerat peste tot cu respect. El observă inimile, diferite una de alta, ale tuturor oamenilor, fără a învinovăți pe nimeni. El este bine versat în toate problemele legate de originea lucrurilor. El nu disprețuiește sau arată ură pentru niciun fel de știință. Trăiește conform propriului său standard de moralitate. El nu își lasă timpul să treacă în zadar. Sufletul lui este sub controlul său. Pentru asta este venerat peste tot cu respect. El a muncit în subiecte care merită aplicarea efortului. A dobândit cunoștințe și înțelepciune. Nu este niciodată sătul de yoga. Este întotdeauna atent și gata de efort. Este mereu vigilent. Pentru asta este venerat peste tot cu respect. El nu are niciodată de simțit rușine pentru vreo deficiență a sa. Este foarte atent. Este întotdeauna angajat de alții în realizarea a ceea ce este pentru binele lor. El nu dezvăluie niciodată secretele altora. Pentru asta este venerat peste tot cu respect. El nu se lasă niciodată pradă bucuriei la ocazii de a face chiar achiziții valoroase. Nu este niciodată dureros la pierderi. Înțelegerea lui este fermă și stabilă. Sufletul lui este neatașat de toate lucrurile. Pentru asta este venerat peste tot cu respect. Cine, într-adevăr, nu îl va iubi pe cel care este astfel înzestrat cu fiecare merit și realizare, care este priceput în toate lucrurile, care este curat în trup și minte, care este în întregime favorabil, care este bine versat cu cursul timpului și cu oportunitatea pentru anumite acte, și care este bine familiarizat cu toate lucrurile plăcute?'"'"


Note:

869. Traducătorul din Burdwan înțelege greșit cuvântul anavajnata. K.P.Singha îl ocolește.

870. Sensul este următorul: deși în realitate nu este atașat, pare a fi atașat. În acest lucru există un merit deosebit. Un om care își îndeplinește îndatoririle de cap de familie, fără a fi însă atașat de soție, copii și bunuri, este o persoană foarte superioară. Un astfel de om a fost comparat cu o frunză de lotus, care, atunci când este scufundată în apă, nu este niciodată înmuiată sau udată de aceasta. Unii, văzând dificultatea luptei, fug. În acest lucru există puțin merit. A înfrunta toate obiectele de dorință, a te bucura de ele, dar totuși a rămâne atât de neatașat de ele încât să nu simți cea mai mică durere dacă te desparți de ele, este mai meritoriu.
