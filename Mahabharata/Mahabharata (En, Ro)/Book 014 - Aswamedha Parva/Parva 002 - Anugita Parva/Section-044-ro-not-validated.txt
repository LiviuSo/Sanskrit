SECTIUNEA 44

"Brahma a spus: 'Vă voi spune acum adevărul despre tot ceea ce are un început, un mijloc și un sfârșit, și care este înzestrat cu nume și caracteristici, împreună cu mijloacele de înțelegere. S-a spus că Ziua a fost prima. Apoi a apărut Noaptea. Se spune că Lunile au avut întâi jumătățile luminate. Constelațiile au Sravana ca primul lor; Anotimpurile au acel lucru al rouelor (adică, Iarna) ca primul lor. Pământul este sursa tuturor mirosurilor; și Apa a tuturor gusturilor. Lumina solară este sursa tuturor culorilor; Vântul a tuturor senzațiilor de atingere. De asemenea, de sunet sursa este spațiul (sau Eterul). Acestea sunt calitățile elementelor. Voi declara, după aceasta, ceea ce este primul și cel mai înalt dintre toate entitățile. Soarele este primul dintre toate corpurile luminate. Focul este considerat primul dintre toate elementele. Savitri este primul dintre toate ramurile învățăturii. Prajapati este primul dintre toate zeitățile. Silaba Om este prima dintre toate Vedele, iar vântul vieții Prana este primul dintre toate vânturile. Tot ceea ce se numește Savitri care este prescris în această lume.[130] Gayatri este prima dintre toate metrurile; dintre toate animalele (sacrificiale) primul este capra. Bovinele sunt primele dintre toate patrupedele. Cei renăscuți sunt primii dintre toți oamenii. Șoimul este primul dintre toate păsările. Dintre sacrificii, primul este turnarea untului clarificat pe foc. Dintre toate reptilele, primul, O cel mai de seamă dintre cei regenerați, este șarpele. Krita este prima dintre toate Yugas; nu există nicio îndoială în acest sens. Aurul este primul dintre toate lucrurile prețioase. Orzul este primul dintre toate plantele. Mâncarea este prima dintre toate lucrurile de mâncat sau înghițit. Dintre toate substanțele lichide de băut, apa este cea mai importantă. Dintre toate entitățile imobile fără distincție, Plaksha este considerată prima, acel câmp sfânt al lui Brahman. Dintre toți Prajapatis eu sunt primul. Nu există nicio îndoială în acest sens. De suflet inconceput, se spune că Vishnu auto-existenta este superiorul meu.[131] Dintre toate munții, se spune că marele Meru este primul născut. Dintre toate punctele cardinale și subsidiare ale orizontului, se spune că estul este cel mai important și primul născut. Ganga cu trei cursuri este considerată prima născută dintre toate râurile. De asemenea, dintre toate fântânile și rezervoarele de apă, se spune că oceanul este primul născut. Iswara este supremul Domn al tuturor zeităților și Danavas și ființelor fantomatice și Pisachas, și șerpi și Makshasas și ființe umane și Kinnaras și Yakshas. Marele Vishnu, care este plin de Brahma, decât care nu există o ființă mai înaltă în cele trei lumi, este primul dintre toate universul. Dintre toate modurile de viață, acela al gospodarului este primul. Nu există nicio îndoială în acest sens. Nemanifestatul este sursa tuturor lumilor, așa cum, într-adevăr, acela este sfârșitul a tot. Zilele se termină cu apusul soarelui și nopțile cu răsăritul soarelui. Sfârșitul plăcerii este întotdeauna durere, iar sfârșitul durerii este întotdeauna plăcere. Toate acumulările au epuizare pentru sfârșitul lor, și toate ascensiunile au căderi pentru sfârșitul lor. Toate asocierile au disocieri pentru sfârșitul lor, iar viața are moarte pentru sfârșitul ei. Toată acțiunea se termină în distrugere, și tot ceea ce se naște este sigur să se întâlnească cu moartea. Fiecare lucru mobil și imobil din această lume este tranzitoriu. Sacrificiu, dar, penitențe, studiu, jurăminte, observanțe, - toate acestea au distrugere pentru sfârșitul lor. Cunoașterea nu are sfârșit. Prin urmare, cel care posedă un suflet liniștit, care și-a supus simțurile, care este eliberat de sentimentul de meum, care este lipsit de egoism, este eliberat de toate păcatele prin cunoaștere pură.'"


Note:

130. După cum a explicat Nilakantha, cuvântul Savitri este folosit aici pentru a implica toate formele de cult observate de Brahmanas, etc, și de Mlecchas de asemenea. Această întoarcere pentru a explica un cuvânt folosit anterior este considerată a fi un exemplu de "privire înapoi ca leul".

131. Cred că Telang traduce greșit acest vers. În prima linie se spune că Brahman este superior Prajapatiilor. În a doua linie se subliniază că Vishnu este superior lui Brahman.
