SECTIUNEA 43

"Brahma a spus: "Dintre oameni, regele Kshatriya este (înzestrat cu) calitatea medie. Dintre vehicule, elefantul (este așa); și dintre locuitorii pădurii, leul; dintre toate animalele (sacrificiale), oaia; dintre toți cei care trăiesc în găuri, este șarpele; dintre vite, taurul bovin; dintre femele, măgarul.[125] Nu există nicio îndoială în acest sens că în această lume, Nyagrodha, Jamvu, Pippala, Salmali și Sinsapa, Meshasringa și Kichaka, sunt cei mai de seamă dintre copaci.[126] Himavat, Patipatra, Sahya, Vindhya, Trikutavat, Sweta, Nila, Bhasa, Koshthavat, Guruskandha, Mahendra și Malayavat, - aceștia sunt cei mai de seamă dintre munți. De asemenea, Maruts sunt cei mai de seamă dintre Ganas. Surya este stăpânul tuturor planetelor, iar Chandramas al tuturor constelațiilor. Yama este stăpânul Pitris; Oceanul este stăpânul tuturor râurilor. Varuna este regele apelor. Indra este considerat regele Maruts. Arka este regele tuturor corpurilor fierbinți, iar Indra al tuturor corpurilor luminoase. Agni este stăpânul etern al elementelor, iar Vrihaspati al Brahmanilor. Soma este stăpânul (ierburilor) căzătoare, iar Vishnu este cel mai de seamă dintre toți cei înzestrați cu putere. Tashtri este regele Rudras, iar Siva al tuturor creaturilor. Sacrificiul este cel mai de seamă dintre toate ritualurile inițiatice, iar Maghavat al zeilor. Nordul este stăpânul tuturor punctelor cardinale; Soma de mare energie este stăpânul tuturor Brahmanilor învățați. Kuvera este stăpânul tuturor pietrelor prețioase, iar Purandara al tuturor zeilor. Aceasta este cea mai înaltă creație dintre toate entitățile. Prajapati este stăpânul tuturor creaturilor. Dintre toate entitățile, eu, care sunt plin de Brahman, sunt cel mai de seamă. Nu există nicio entitate care este mai înaltă decât mine sau Vishnu. Marele Vishnu, care este plin de Brahman, este regele regilor peste toți. Cunoașteți-l ca pe conducător, creator, Hari necreat. El este conducătorul oamenilor și Kinnaras și Yakshas și Gandharvas, și Șerpi și Rakshasas, și zei și Danavas și Nagas. Dintre cei care sunt urmați de persoane pline de dorință este marea zeiță Maheswari cu ochi frumoși. Ea este cunoscută și sub numele de Parvati. Să știi că zeița Uma este cea mai de seamă și cea mai favorabilă dintre femei. Dintre femeile care sunt o sursă de plăcere, cele mai de seamă sunt Apsaras care sunt înzestrate cu mare strălucire.[127] Regii doresc să dobândească pietatea, iar Brahmanii sunt căi de pietate. Prin urmare, regele ar trebui întotdeauna să se străduiască să protejeze pe cei născuți de două ori. Acei regi în a căror domenii oamenii buni se sting sunt considerați lipsiți de virtuțile ordinii lor. În viitor, ei trebuie să meargă pe căi greșite. Acei regi în a căror domenii oamenii buni sunt protejați, se bucură în această lume și se bucură de fericire în viitor. Într-adevăr, acei oameni cu suflet înalt ajung la cel mai înalt scaun. Înțelegeți acest lucru, voi cei mai de seamă dintre cei regenerați. Voi declara după aceasta indicațiile eterne ale datoriilor. Abținerea de la rănire este cea mai înaltă datorie. Rănirea este o indicație a nedreptății. Strălucirea este indicația zeilor. Oamenii au acte pentru indicațiile lor. Eterul (sau spațiul) are sunetul ca caracteristică. Vântul are atingerea ca caracteristică. Caracteristica corpurilor luminate este culoarea, iar apa are gustul ca caracteristică. Pământul, care deține toate entitățile, are mirosul ca caracteristică. Discursul are cuvintele ca caracteristică, rafinate în vocale și consoane. Mintea are gândul ca caracteristică. S-a spus că gândul este caracteristica înțelegerii. Lucrurile la care se gândește mintea sunt stabilite cu precizie de înțelegere. Nu există nicio îndoială în acest sens, adică înțelegerea, prin perseverență, percepe toate lucrurile. Caracteristica minții este meditația. Caracteristica omului bun este să trăiască neobservat.[128] Devotamentul are acte ca caracteristică. Cunoașterea este caracteristica renunțării. Prin urmare, păstrând cunoașterea în fața sa, omul de înțelegere ar trebui să practice renunțarea. Omul care s-a dedicat renunțării și care este înzestrat cu cunoaștere, care transcende toate perechile de opuse, precum și întunericul, moartea și decrepitudinea, atinge obiectivul cel mai înalt. V-am declarat astfel în mod corespunzător care sunt indicațiile datoriei. Voi spune după aceasta despre captarea (înțelegerea) calităților. Mirosul, care aparține pământului, este captat de nas. Vântul, care locuiește în nas, este de asemenea numit (ca agent) în percepția mirosului. Gustul este esența apei. Acesta este captat de limbă. Soma, care locuiește în limbă, este numit de asemenea în percepția gustului. Calitatea unui corp luminat este culoarea. Aceasta este captată de ochi. Aditya, care locuiește întotdeauna în ochi, a fost numit în percepția culorii. Atingerea aparține întotdeauna vântului (ca și calitatea sa). Aceasta este percepută de piele. Vântul care locuiește întotdeauna în piele a fost numit în înțelegerea atingerii. Calitatea eterului este sunetul. Acesta este captat de ureche. Toate direcțiile, care locuiesc în ureche, au fost numite în înțelegerea sunetului. Calitatea minții este gândul. Acesta este captat de înțelegere. Susținătorul conștiinței, care locuiește în inimă, a fost numit în înțelegerea minții. Înțelegerea este înțeleasă sub forma hotărârii sau certitudinii, iar Mahat sub forma cunoașterii. Nepătrunsul (Prakriti) a fost, este evident, numit pentru captarea tuturor lucrurilor după certitudine. Nu există nicio îndoială în acest sens.[129] Kshetrajna, care este etern și este lipsit de calități în ceea ce privește esența sa, este incapabil de a fi captat prin simboluri. Prin urmare, caracteristica Kshetrajna, care este fără simboluri, este pur și simplu cunoașterea. Nepătrunsul locuiește în simbolul numit Kshetra și este acela în care calitățile sunt produse și absorbite. Întotdeauna îl văd, îl cunosc și îl aud (deși) este ascuns. Purusha îl cunoaște: de aceea este numit Kshetrajna. Kshetrajna percepe de asemenea operațiunile calităților și absența operațiunilor lor. Calitățile, care sunt create în mod repetat, nu se cunosc pe ele însele, fiind neinteligente, ca entități care urmează să fie create și înzestrate cu un început, mijloc și sfârșit. Nimeni altcineva nu atinge, doar Kshetrajna atinge, ceea ce este cel mai înalt și mare și care transcende calitățile și acele entități care sunt născute din calități. Prin urmare, cel care înțelege datoriile, renunțând la calități și la înțelegere, și având păcatele distruse, și transcendent calitățile, intră în Kshetrajna. Cel care este liber de toate perechile de opuse, care nu-și pleacă niciodată capul în fața nimănui, care este lipsit de Swaha, care este imobil și fără adăpost, este Kshetrajna. El este Domnul Suprem."


Note:

125. Ceea ce se spune aici este că predomină calitatea pasiunii în acestea.

126. Nyagrodha este Ficus Bengalensis, Linn. Jamvu este Eugenia Jambolana, Lamk. Pippala este Ficus religiosa, Linn. Salmali este Bombax Malabaricum. Sinsapa este Dalbergia Sissoo, Roxb. Meshasringa este Asclepias geminata, Roxb. Kichaka este o varietate de bambus de munte. Aici, însă, se înțelege în mod evident Nimba sau Melia Azadirachta, Linn.

127. Nilakantha este pentru a lua a doua linie ca fiind compusă din două propoziții. Ar fi mai bine să luăm satinam ca referindu-se la strinam și vasumatyah, ca un adjectiv al Apsarasah.

128. Sensul pare a fi acela că oamenii buni nu permit niciodată altora să știe care sunt acțiunile lor. Ei sunt străini față de ostentație.

129. Sensul pare a fi acela că cunoașterea propriei identități și a lucrurilor diferențiate una de cealaltă este prezidată de Prakriti. Dacă se pune întrebarea de unde vine cunoașterea - 'Eu sunt așa', și că 'acesta este așa', răspunsul este că vine de la Prakriti sau Natura.
