SECTIUNEA 24

"Brahmana a spus, 'În această legătură este citită povestea antică a discursului dintre Narada și Rishi Devamata.'"

Devamata a spus: "Ce apare cu adevărat prima dată la existența unei creaturi care se naște? Este Prana sau Apana sau Samana sau Vyana sau Udana?"

Narada a spus: "Cu orice creatură este creată, aceasta vine întâi la el, care este altul (sau separat de el). Vânturile vieții trebuie cunoscute ca existând în perechi, adică cele care se mișcă transversal, în sus și în jos."

Devamata a spus: "De cine (dintre vânturile vieții) este produsă o creatură? Cine (dintre ei) vine primul? Spune-mi care sunt perechile vânturilor vieții, care se mișcă transversal, în sus și în jos."

"Narada a spus: 'Din Sankalpa (dorință) se naște Plăcerea. Aceasta se naște și din sunet. Se naște și din gust; se naște și din culoare. Din sperma, unită cu sângele, curge prima dată Prana. După ce sperma este modificată de Prana, curge Apana. Plăcerea se naște și din spermă. Se naște și din gust. Aceasta este forma (efectul) a Udana. Plăcerea este produsă din uniune. Sperma este generată de dorință. Din dorință este produs fluxul menstrual. În uniunea spermei și a sângelui, generată de Samana și Vyana, perechea care constă în Prana și Apana, intră, mișcându-se transversal și în sus, Vyana și Samana formează ambele o pereche care se mișcă transversal. Agni (focul) este toate zeitățile. Chiar aceasta este învățătura Vedei. Cunoașterea lui Agni apare într-un Brahmana cu inteligență. Fumul acelui foc este de forma (atributului numit) Întuneric. Atributul care este cunoscut sub numele de Pasiune este în cenușa sa. Calitatea bunătății se naște din acea porțiune a focului în care este turnată ofranda.[66] Cei care sunt familiarizați cu sacrificiile știu că Samana și Vyana provin din atributul Bunătății. Prana și Apana sunt părți ale ofrandei (de unt clarificat). Între ele este Focul. Aceasta este forma excelentă (sau locul) a Udana, așa cum știu Brahmanii. Ascultați cum spun care este distinctă de perechi. Ziua și Noaptea constituie o pereche. Între ele este Focul. Acesta este locul excelent al Udana așa cum știu Brahmanii. Existentul și non-existentul formează o pereche. Între ele este Focul. Acesta este locul excelent al Udana așa cum știu Brahmanii. Primul este Samana; apoi Vyana. Funcția acestuia din urmă este gestionată prin ea (adică, Samana). Apoi, în al doilea rând, Samana intră din nou în funcțiune. Doar Vyana există pentru liniște. Liniștea este Brahmanul etern. Acesta este locul excelent al Udana așa cum știu Brahmanii.'"[67]


Note:

66. Citirea sarvam din a doua linie este incorectă, deși Nilakantha o adoptă. Diferitele părți ale focului sunt indicate ca fiind diferite atribute. Fumul este în forma Întunericului (Tamas); cenușa sunt atributele Pasiunii; în timp ce flacăra aprinsă, în care este aruncată ofranda, este atributul Bunătății.

67. Ofer o interpretare apropiată a acestor versuri, fără a încerca să aduc la suprafață sensul așa cum este explicat de comentatori. Textele tipărite nu sunt corecte. Textul adoptat de Nilakantha diferă de cel al lui Arjuna Misra. Chiar ordinea versurilor nu este uniformă în toate textele.
