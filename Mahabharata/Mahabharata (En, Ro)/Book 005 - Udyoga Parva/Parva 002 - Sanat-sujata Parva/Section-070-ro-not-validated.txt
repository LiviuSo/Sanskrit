SECTIUNEA 70

"Dhritarashtra a spus, 'Te rog, O Sanjaya, să-mi vorbești din nou despre Krishna cu ochii de lotus; pentru că, fiind familiarizat cu semnificația numelui său, aș putea, O fiule, să obțin acel cel mai înălțat dintre ființele masculine.'"

"Sanjaya a spus: 'Numele favorabile (ale lui Kesava) mi-au fost auzite anterior. Din acestea, îți voi spune câte cunosc. Kesava, totuși, este nemăsurabil, fiind peste puterea cuvântului de a descrie. El este numit Vasudeva ca urmare a faptului că învăluie toate creaturile cu ecranul iluziei, sau a strălucirii sale glorioase, sau a faptului că este suportul și locul de odihnă al zeilor. El este numit Vishnu din cauza naturii sale omniprezente. El este numit Madava, O Bharata, din cauza practicării sale ca Muni, concentrarea minții asupra adevărului și absorbția Yoga. El este numit Madhusudana din cauza faptului că a ucis Asura Madhu, și de asemenea din cauza faptului că este substanța celor douăzeci și patru de obiecte de cunoaștere. Născut din rasa Sattwata, el este numit Krishna pentru că unește în sine ceea ce este implicat de cele două cuvinte Krishi care înseamnă "ceea ce există" și na care înseamnă "pace eternă". El este numit Pundarikaksha din Pundarika care implică locuința sa înaltă și eternă, și Aksha care înseamnă "indestructibil"; și el este numit Janardana pentru că inspiră frică în inimile tuturor ființelor rele. El este numit Sattwata, pentru că atributul Sattwa nu este niciodată disociat de el și de asemenea pentru că el nu este niciodată disociat de acesta; și el este numit Vrishabhakshana din Vrishabha care implică "Vedele" și ikshana care implică "ochi", uniunea celor două semnificând că Vedele sunt ochii lui, sau Vedele sunt ochii prin care el poate fi văzut. Acel cuceritor de oști este numit Aja, sau "nenăscut", pentru că nu și-a luat nașterea de la nicio ființă în mod obișnuit. Acel suflet suprem este numit Damodara pentru că, spre deosebire de zei, strălucirea sa este increată și proprie, și de asemenea pentru că are autocontrol și o mare strălucire. El este numit Hrishikesa, din Hrishika care înseamnă "fericire eternă" și Isa care înseamnă "cele șase atribute divine", uniunea semnificând unul care are bucurie, fericire și divinitate. El este numit Mahavahu, pentru că susține pământul și cerul cu cele două brațe ale sale. El este numit Adhakshaja, pentru că nu cade niciodată sau nu suferă nicio deteriorare, și este numit Narayana pentru că este refugiul tuturor ființelor umane. El este numit Purusottama din Puru care implică "cel care creează și păstrează" și astfel înseamnă "cel care distruge, uniunea semnificând unul care creează, păstrează și distruge universul". El posedă o cunoaștere a tuturor lucrurilor și, prin urmare, este numit Sarva. Krishna este întotdeauna în Adevăr și Adevărul este întotdeauna în el, iar Govinda este Adevărul Adevărului. Prin urmare, el este numit Satya. El este numit Vishnu din cauza puterii sale și Jishnu din cauza succesului său. El este numit Ananta din cauza eternității sale și Govinda din cauza cunoașterii sale despre discursul de orice fel. El face ca irealul să apară ca real și astfel înșală toate creaturile. Posedând astfel de atribute, mereu devotat dreptății și înzestrat cu divinitate, ucigașul lui Madhu, acel puternic înarmat incapabil de decădere, va veni aici pentru a preveni masacrarea Kurusilor.'"
